

Запуск проекта, открыть лучше в pycharm, команды вводить в терминале

1 - Установить необходимые файлы с помощью команды

    pip install -r requirements.txt
 
2 - Запустить сервер на 8000 порту, для этого ввести команду

    python manage.py runserver
    
    и в браузере вписать:

    http://127.0.0.1:8000/
    
3 - Создать суперпользователя командой

    python manage.py createsuperuser
